# Adventure-works
# Project Overview
This project analyzes Adventure Works Cycles, a global bicycle manufacturing company, with the goal of expanding market share and reducing operational costs. Using SQL, Excel, Power BI, and Tableau, the project delivers insights into customer behavior, product performance, regional sales trends, and profitability.

# Problem Statement
Adventure Works lacked clear visibility into:
High-value customers and their purchasing behavior
Top-performing products and revenue contributors
Regional sales performance across global markets
Cost inefficiencies in production and distribution

This project addresses these challenges through structured data modeling and interactive dashboards.

# Data Sources
Adventure Works Excel dataset
Fact and Dimension tables (Sales, Customers, Products, Geography)
SQL queries for integration, transformation, and calculated fields

#Data Processing and Analysis (SQL)
Integration: UNION ALL to consolidate sales tables
Transformations: Extracted Year, Month, Quarter, Weekday; created Financial Month and Quarter
Calculated Columns: Sales Amount, Production Cost, Profit
Aggregations: Year-wise, Month-wise, Quarter-wise performance

# Dashboard Development Process
Data Collection and Cleaning – removed duplicates, handled missing values, formatted data
Data Preparation and Linking – structured tables, relationships via common fields
Calculations – Total Sales, Cost, Profit, Profit Margin %
Visualization Design – KPI cards, bar charts, line charts, slicers
Insights and Analysis – identified trends, top-performing regions/products, seasonal patterns

# Conclusion
Through SQL-based data processing and BI dashboards in Excel, Power BI, and Tableau, this project provided actionable insights to:
Increase market share
Optimize operational costs
Target high-value customers effectively

# Project Impact
Processed and analyzed 60K+ orders and 29M+ sales data
Built dashboards covering 18.5K customers across multiple regions
Delivered insights that highlight top products, seasonal trends, and profitability drivers
